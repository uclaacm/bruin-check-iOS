# bruin-check-iOS


## INSTALLATION  
  
Download CocoaPods:  
```
$ sudo gem install cocoapods
```  
To check if the installation succeeds or the version of CocoaPods:  
```
$ gem which cocoapods  
```
After clone the project,  
```
$ cd $PATH/BruinCheck-iOS-End  
$ pod install
$ open BruinCheck-iOS-End.xcworkspace
```
